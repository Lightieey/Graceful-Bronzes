# 🤎Graceful-Bronzes🤎
> **우아한 브론즈들** : 개발자 기업 '우아한 형제들'과 백준의 티어 '브론즈'를 합하여 지어진 팀명  
비록 백준 티어에서 가장 낮은 브론즈들이지만, 열심히 공부하여 브론즈를 벗어나 훌륭한 코더가 되고자 하는 의미를 담고 있다.  
- 기간 : 2022년 9월 24일 ~
- 정기회의 : 토요일 아침 10시
- 참고사이트 : [백준-단계별로 풀어보기](https://www.acmicpc.net/step)
- 언어 : 파이썬, c++
